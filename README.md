# Política de Privacidade – Yunæra

Yunæra coleta apenas dados necessários para funcionar corretamente, incluindo:
- IDs de usuários
- IDs de servidores
- Comandos enviados ao bot

Nenhuma informação pessoal é vendida, compartilhada ou usada para fins externos.

Os dados são usados apenas para as funcionalidades internas do bot.

Última atualização: 2025
